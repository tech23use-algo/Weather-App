# Weather App

A responsive weather application built using HTML, CSS and JavaScript that fetches real-time weather data from the OpenWeatherMap API.

## Features

- Search weather by city
- Real-time temperature
- Humidity
- Wind speed
- Weather description
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript 
- OpenWeatherMap API

## What I Learned

- DOM Manipulation
- Fetch API
- Async/Await
- JSON parsing

## Note

The HTML and CSS files are enhanced using AI assistance as a result of which The visual design of the app is enhanced.The JavaScript logic is written by me as a part of my understanding and AI assistance was used wherever required while writing the code.
